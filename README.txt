# 🎉 OrçaFácil Pro v2 🚀

Bem-vindo ao **OrçaFácil Pro v5**, o app que vai transformar a forma de gerenciar suas vendas e despesas! 💸💼  


---

## ✨ Funcionalidades Principais

➡️ **Login e Cadastro de Usuários** 🔐  
- Crie sua conta e faça login com segurança!  
- Use o mesmo usuário para acessar suas vendas e relatórios.  

➡️ **Gerenciamento de Vendas** 💰  
- Adicione vendas rapidamente pelo Dashboard.  
- **Cancele** vendas com erro ❌  
- **Apague** vendas caso não queira registrar 🗑️  

➡️ **Relatórios em PDF** 📝  
- Gere relatórios completos com gráfico incluso.  
- Salve ou compartilhe com sua equipe facilmente.  

---

## 🛠️ Instalação Passo a Passo

1️⃣ **Clone o repositório**:  
```bash
git clone https://github.com/seu-usuario/orcafacil-pro-v5.git
cd orcafacil-pro-v5
```

2️⃣ **Instale as dependências**:  
```bash
pip install -r requirements.txt
```

3️⃣ **Execute o app**:  
```bash
python app.py
```

4️⃣ **Acesse no navegador**:  
```
http://127.0.0.1:5000
```


---

## 😎 Dicas Extras

- Personalize os templates em `templates/` para deixar o app com a sua cara. 🎨  
- Atualize o CSS em `static/login_style.css` para cores e animações. 🌈  
- Use sempre Python 3.10+ e mantenha o banco SQLite seguro. 🔐
